# Shadow Big LED

## 概要
本プロジェクトは、OpenGL (GLUT) および OpenCV を用いて実装されたインタラクティブな影のシミュレーションシステムです。仮想空間上の物体に対する光源（LEDパネル）の位置を変化させることで生成される影をリアルタイムでシミュレーションします。また、シリアル通信を介して LiDAR 等の物理センサーと連携することで、現実空間でのユーザの動き（手や足のタッチ）をトラッキングし、仮想空間の影を直接操作できるインタラクティブな体験を実現します。

## 主な機能
- **リアルタイム3D描画と影のシミュレーション**: 物体（テクスチャ板、立方体、球体など）に対し、指定の光源からあたる光による影を計算・描画します。
- **インタラクティブな操作**:
  - LiDAR データやマウスを用いたタッチ・ドラッグによる影の移動および光源の再計算処理。
  - ノイズ低減の移動平均フィルタを通して、安定したトラッキングを実施。
- **多光源と色付き影へのグラフィカルな対応 (`main2.cpp`)**: 
  - 複数の光源を用いた際に生じる「赤い影」「緑の影」「重なり合った影」の領域をプログラム(画像処理)ベースで識別。
  - 触れた影の色に応じて、各独立した光源に対する操作をリアルタイムに分岐。
- **LED等へのマッピング視点出力**: シミュレーション結果と現実のLEDパネル等の出力を連携させるための専用ウィンドウ（`CG1`, `CG2`等）を生成します。

## 依存ライブラリと動作環境
本システムは macOS での開発・実行を想定して設計されています。
- **C++11対応コンパイラ** (g++ 等)
- **OpenGL / GLUT フレームワーク** (macOS 標準フレームワーク)
- **OpenCV 4** (`pkg-config` でリンクパスを取得)
- シリアル通信設定 (UNIX の `<termios.h>` 等によるシリアル通信機能)

## 主要ファイル
- `main01.cpp`: 単一の光源（巨大なLEDパネル）を想定した標準的な影トラッキング用メインプログラム。
- `main01_Scan.cpp`: `main01.cpp` をベースに、PLY形式の3Dモデル（スキャンデータ等）の読み込み・描画に対応した拡張プログラム。
- `main2.cpp`: 2つの光源による複数・多色の影（赤・緑・重なり部分）を個別に追従・操作させるためのマルチライト対応版プログラム。
- `Obj_*.png` 等: アプリ内で利用されるテクスチャ用の画像素材。

## コンパイルと実行方法

ターミナルで該当ディレクトリに移動後、以下のコマンドでコンパイルおよび実行を行います。環境変数等で `pkg-config` が正しく認識されている必要があります。

### `main01.cpp` をコンパイルする場合
```bash
g++ -O3 -std=c++11 main01.cpp -framework OpenGL -framework GLUT `pkg-config --cflags --libs opencv4` -Wno-deprecated
./a.out
```

### `main01_Scan.cpp` をコンパイルする場合
```bash
g++ -O3 -std=c++11 main01_Scan.cpp -framework OpenGL -framework GLUT `pkg-config --cflags --libs opencv4` -Wno-deprecated
./a.out
```

### `main2.cpp` をコンパイルする場合
```bash
g++ -O3 -std=c++11 main2.cpp -framework OpenGL -framework GLUT `pkg-config --cflags --libs opencv4` -Wno-deprecated
./a.out
```

> **Note**: アプリケーション起動時、シリアルポート(`"/dev/cu.usbserial-10"` など)が見つからない場合はエラーが標準出力に表示されますが、描画処理のみの単独動作も可能です。実機接続時はソースコード内のポート名を環境に合わせて変更してください。

## アプリケーションの操作方法（キー定義）
描画ウィンドウがアクティブな状態で、以下のキー操作に対応しています。

- `d` : 通常モードとデバッグ用ガイド表示（光源位置・座標軸等の表示）の切り替え
- `r` : カメラ視点（CG1 ウィンドウ）のリセット
- `t` : 光源位置の初期位置リセット
- `0` : 投影する物体を「テクスチャ板」に変更
- `1` : 投影する物体を「立方体」に変更
- `2` : 投影する物体を「立方体と球体」に変更
- `f` : フルスクリーンモードへの切り替え
- `ESC` : プログラムの終了
